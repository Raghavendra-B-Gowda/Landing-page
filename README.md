# 🌐 Landing Page

A modern and responsive landing page built using HTML and CSS. This project showcases a clean UI design with multiple sections like Hero, Features, About, Services, and Footer.

## 🚀 Features
- Responsive design (mobile, tablet, desktop)
- Smooth navigation
- Clean and modern UI
- Reusable components

## 🛠️ Technologies Used
- HTML5
- CSS3
- Google Fonts
- Font Awesome
